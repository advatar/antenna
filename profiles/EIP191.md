
# EIP-191 Profile: Sign bytes32 eventHash

This repo uses a minimal, deterministic EIP-191 profile.

## Message
The message is exactly 32 bytes: the event hash bytes of `event.id` (SHA-256 output).

## Digest
`digest = keccak256("\x19Ethereum Signed Message:\n32" || eventHashBytes32)`

## Signature encoding
`signatureHex = 0x{r}{s}{v}` (65 bytes), with `v` in `{27,28}`.
