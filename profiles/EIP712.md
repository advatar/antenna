
# EIP-712 Profile: MBEvent (Minimal)

This repo defines a *minimal* EIP-712 profile used in interop vectors.

## Domain
Type string:

`EIP712Domain(string name,string version,uint256 chainId,address verifyingContract)`

Values:
- `name = "AntennaEvent"`
- `version = "1"`
- `chainId = <EVM chain id>`
- `verifyingContract = <ERC-8004 IdentityRegistry address>`

## Primary type

`MBEvent(bytes32 eventHash)`

Where:
- `eventHash` is the 32-byte event id hash (SHA-256 output), i.e. the bytes of `event.id`.

## Digest
`digest = keccak256("\x19\x01" || domainSeparator || structHash)`

## Signature encoding
`signatureHex = 0x{r}{s}{v}` (65 bytes), with `v` in `{27,28}`.

Reference implementation:
- `python/antenna_spec_tools/secp256k1.py`
